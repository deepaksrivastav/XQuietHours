XQuietHours

This application was developed for personal use and hence has been tested only on a Nexus 5 with Lollipop AOSP ROMs.

Code from the following projects were referred for the development of this application.

  • AOSP (http://source.android.com/)
  • CyanogenMod (https://github.com/CyanogenMod)
  • GravityBox KitKat Module(https://github.com/GravityBox)
  • WhatsappQuietHours (https://github.com/pbosio/WhatsappQuietHours)

Due credits to the authors of the above projects.
